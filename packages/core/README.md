# Commerce Core

[![Latest Version on Packagist](https://img.shields.io/packagist/v/jeffersongoncalves/laravel-commerce-core.svg?style=flat-square)](https://packagist.org/packages/jeffersongoncalves/laravel-commerce-core) [![Total Downloads](https://img.shields.io/packagist/dt/jeffersongoncalves/laravel-commerce-core.svg?style=flat-square)](https://packagist.org/packages/jeffersongoncalves/laravel-commerce-core) [![License](https://img.shields.io/packagist/l/jeffersongoncalves/laravel-commerce-core.svg?style=flat-square)](LICENSE.md)

Commerce core — base service, saga workflow engine, prefixed identifiers, and money primitives for the Laravel Commerce ecosystem

## Installation

```bash
composer require jeffersongoncalves/laravel-commerce-core
```

## Usage

The service provider is auto-discovered. Publish migrations and use the module models and services directly. See the [umbrella package](https://github.com/jeffersongoncalves/laravel-commerce) for the full ecosystem.
